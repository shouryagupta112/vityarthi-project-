## 📂 Project Description
This project analyzes the open-source software **Git** from technical, ethical, and system-level perspectives. It also includes shell scripts developed on a Linux (WSL Ubuntu) environment.

---

## 📜 Scripts Included

### 1. script1.sh — System Identity Report
Displays:
- Kernel version  
- Current user  
- System uptime  
- Date and time  

---

### 2. script2.sh — Package Inspector
Checks:
- Whether Git is installed  
- Displays version and package details  

---

### 3. script3.sh — Disk and Permission Auditor
Analyzes:
- Important system directories  
- Permissions, owner, and size  

---

### 4. script4.sh — Log File Analyzer
- Reads log files  
- Counts occurrences of a keyword (default: "error")  

---

### 5. script5.sh — Manifesto Generator
- Takes user input  
- Generates an open-source manifesto in a text file
